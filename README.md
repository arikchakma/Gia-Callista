# Gia-Callista
It's a javascript utilities library for working fast &amp; we're making it better day by day. 

# Instalation
1. Using node package manager:
    ```shell
    npm i gia-callista
    ```
2. Using yarn
    ```shell
    yarn add gia-callista
    ```
