# Gia-Callista

It is a javascript utility package for coding fast, and we are making it tasty day by day.

# Instalation

1. Using node package manager:
   ```shell
   npm i gia-callista
   ```
2. Using yarn
   ```shell
   yarn add gia-callista
   ```
